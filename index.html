About This Calculator

Welcome to the **Midgard Barter Calculator**, an unofficial tool designed to accurately assess the value of trades based on community-sourced LS (Life Stone) prices and complex in-game mechanics. This tool is inspired by and credits **Solarius Sensei** for the extensive data collection. Visit Solarius Sensei's YouTube Channel.

How to Use the Calculator

  1. **Target Item:** Search for the item you wish to acquire on the left panel and set the desired **Quantity**. Its final value is locked.
  2. **Offer Items:** Use the mixer in the center to select up to five items you are offering.
  3. **Gear Rules:** If you select a **Weapon** or **Armor Set**, dynamic controls will appear:
    • **Upgrade Slider:** Automatically limits to the max level (+3, +5, or +10) based on rarity.
    • **Broken Toggle:** Use the switch to apply the **50% LS reduction** for broken gear.
    • **Armor Selector:** For full sets, select a specific piece (Head, Chest, Pants, Boots) to apply the correct weighting (20%, 30%, 22%, 28%).
  4. **Trade Results:** The right panel shows the **Live Difference** and the visual **Balance Bar**. The final verdict provides the exact deficit or suggests refund items.
  5. **Refund Logic:** If you overpay, the calculator first suggests acquiring more of the **Target Item** (if non-gear consumable) before suggesting other filler items.
